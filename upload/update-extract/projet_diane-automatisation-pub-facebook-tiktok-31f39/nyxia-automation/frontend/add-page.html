📖 Comment obtenir vos tokens d'accès

📘 Facebook Graph API

  1. Allez sur developers.facebook.com
  2. Créez une application ou utilisez une existante
  3. Ajoutez le produit "Graph API"
  4. Allez dans "Tools & Support" → "Graph API Explorer"
  5. Sélectionnez votre application et page
  6. Demandez les permissions: pages_manage_posts,pages_read_engagement
  7. Copiez le "User Token" ou "Page Access Token"

📚 Documentation complète: Graph API Docs

🎵 TikTok API

  1. Allez sur developers.tiktok.com
  2. Créez un compte développeur et une application
  3. Dans le dashboard, allez dans "Products" → "Video Upload"
  4. Soumettez votre demande d'accès (approbation requise)
  5. Une fois approuvé, générez un "Access Token" via OAuth 2.0
  6. Utilisez le scope: video.upload
  7. Copiez le token généré

📚 Documentation complète: TikTok API Docs

⚠️ Important: Les tokens Facebook expirent généralement après 60 jours. Vous devrez les renouveler périodiquement. Les tokens TikTok peuvent avoir une durée de vie plus longue selon votre configuration OAuth.
